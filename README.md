## Try it now!

### Online website

See [Link](https://wudao.netlify.app)

### Clone to local

```bash
pnpm install # If you don't have pnpm installed, run: npm install -g pnpm
```

## Usage

### Development

Just run and visit http://localhost:3000

```bash
pnpm dev
```

### Build

To build the App, run

```bash
pnpm build
```
